# Compara Precios API

## Description

API to compare prices of products from different stores.

## Setup

- Clone the repository
- Install the dependencies

```bash
yarn install
```

- Create a `.env` file in the root of the project with the following content:

```bash
cp .env.example .env
```

```env
PORT=3000
```

- Run the server

```bash
yarn dev
```
